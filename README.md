# Async Downloader

An async downloader with python and asyncio.

#

# How to Run Project

## Download Codes

```
git clone https://github.com/dori-dev/downloader.git
```

```
cd downloader
```

## Build Virtual Environment

```
python -m venv env
```

```
source env/bin/activate
```

## Install Project Requirements

```
pip install -r requirements.txt
```

## Download File

```bash
python main.py url
```

## How to Use

```bash
python main.py -h
```
